# Movilidad_Urbana
Proyecto de Bootcamp TripleTen, en el que generé un reporte para comprender cómo la movilidad urbana se relaciona con la productividad económica en las principales ciudades latinoamericanas. La finalidad fue identificar en cuáles ciudades se debe invertir en infraestructura de transporte para aumentar la productividad y el bienestar de la población
